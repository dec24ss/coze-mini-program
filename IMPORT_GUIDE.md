# 海海拼图大作战 - 微信小程序导入指南

## 📦 压缩包信息

- 文件名：`puzzle-game-weapp.tar.gz`
- 大小：21 MB
- 包含内容：完整的微信小程序编译文件

## 🚀 快速开始

### 第一步：解压压缩包

在本地解压压缩包：

```bash
tar -xzf puzzle-game-weapp.tar.gz
```

解压后会得到一个目录，包含以下文件：

```
puzzle-game-weapp/
├── project.config.json    # 微信小程序项目配置
├── app.json               # 小程序配置
├── app.js                 # 小程序入口
├── README.md              # 详细说明文档
├── pages/                 # 页面文件
│   ├── loading/           # 加载页
│   ├── index/             # 首页
│   ├── game/              # 游戏页
│   ├── level-select/      # 关卡选择页
│   └── rank/              # 排行榜页
├── vendors.js             # 依赖库
├── taro.js                # Taro 框架
└── ...                    # 其他文件
```

### 第二步：导入微信开发者工具

1. 打开微信开发者工具
2. 点击"+"号或"导入项目"
3. 选择解压后的 `puzzle-game-weapp` 目录
4. **重要**：输入你的微信小程序 AppID（或使用测试号）
5. 项目名称：`海海拼图大作战`
6. 点击"导入"

### 第三步：配置后端服务（必须）

⚠️ **注意：此小程序必须配置后端 API 服务才能正常运行**

#### 方法一：使用本地开发环境

1. 修改源代码中的后端 API 地址
2. 启动本地后端服务
3. 重新编译小程序

#### 方法二：使用已部署的后端服务

如果已有后端服务，需要修改编译前的源代码：

1. 找到 `src/network/index.ts` 文件
2. 修改 `BASE_URL` 为你的后端服务地址
3. 重新编译微信小程序：
   ```bash
   pnpm build:weapp
   ```

#### 方法三：联系开发者获取后端服务

如果没有后端服务，请联系开发者获取：
- 后端服务地址
- API 文档
- 访问凭证

### 第四步：配置微信小程序

在微信开发者工具中：

1. 点击右上角"详情"
2. 配置以下设置：
   - **本地设置**：
     - 不校验合法域名、web-view（业务域名）、TLS 版本以及 HTTPS 证书（勾选）
     - 不校验合法域名、web-view（业务域名）、TLS 版本以及 HTTPS 证书（勾选）

### 第五步：测试功能

在微信开发者工具中测试以下功能：

1. ✅ 小程序是否能正常启动
2. ✅ 加载页是否正常显示
3. ✅ 首页是否能正常进入
4. ✅ 点击"开始游戏"是否能进入关卡选择
5. ✅ 选择关卡是否能进入游戏页
6. ✅ 微信登录是否正常（需要配置后端）

### 第六步：上传审核

功能测试通过后：

1. 在微信开发者工具中点击"上传"
2. 填写版本号（如：1.0.0）
3. 填写项目备注（如：初始版本）
4. 点击"上传"
5. 等待上传完成

在微信小程序管理后台：

1. 进入"版本管理"
2. 找到刚上传的版本
3. 点击"提交审核"
4. 填写审核信息
5. 提交审核

## ⚙️ 微信小程序管理后台配置

### 1. 配置服务器域名

在"开发" -> "开发管理" -> "开发设置"中配置：

- **request 合法域名**：你的后端 API 域名（如：https://api.example.com）
- **uploadFile 合法域名**：你的后端 API 域名
- **downloadFile 合法域名**：你的后端 API 域名

**注意**：
- 必须使用 HTTPS 协议
- 域名必须备案
- 测试阶段可以勾选"不校验合法域名"

### 2. 配置 AppID

在 `project.config.json` 中修改：

```json
{
  "appid": "你的真实AppID"
}
```

### 3. 配置隐私协议

1. 进入"设置" -> "基本设置"
2. 配置用户隐私保护指引
3. 配置用户信息保护指引

## 🔧 常见问题

### Q1: 小程序启动后显示空白

**原因**：未配置后端 API 服务

**解决**：
- 配置后端 API 地址
- 重新编译小程序

### Q2: 点击按钮没有反应

**原因**：可能是网络请求失败

**解决**：
- 检查后端服务是否正常运行
- 检查网络配置
- 查看控制台错误信息

### Q3: 图片加载失败

**原因**：图片源地址不可用

**解决**：
- 检查图片源（Lorem Picsum）
- 配置代理或更换图片源

### Q4: 微信登录失败

**原因**：未配置后端登录接口

**解决**：
- 配置后端登录 API
- 检查 AppID 配置
