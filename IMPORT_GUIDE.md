# 微信小程序导入指南

## 压缩包信息

- **文件名**: `puzzle-game-miniprogram.tar.gz`
- **文件位置**: `/workspace/projects/puzzle-game-miniprogram.tar.gz`
- **文件大小**: 191KB
- **打包时间**: 2025-03-05

## 使用方法

### 方法一：直接导入（推荐）

1. **获取压缩包**
   - 压缩包位于：`/workspace/projects/puzzle-game-miniprogram.tar.gz`
   - 请通过适当的方式（如 SFTP、SCP 等）将此文件下载到本地

2. **解压压缩包**
   ```bash
   tar -xzf puzzle-game-miniprogram.tar.gz
   ```
   或使用解压软件（如 7-Zip、WinRAR）解压

3. **导入微信开发者工具**
   - 打开微信开发者工具
   - 点击 "+" 号导入项目
   - 选择解压后的 `puzzle-game-miniprogram` 目录
   - 填写 AppID（如果没有，可以选择"测试号"或使用自己的 AppID）

### 方法二：直接使用源码

如果无法下载压缩包，可以直接使用项目源码：

1. **进入项目目录**
   ```bash
   cd /workspace/projects
   ```

2. **构建生产版本**
   ```bash
   pnpm build:weapp
   ```

3. **复制构建文件**
   ```bash
   # 复制构建文件到新目录
   mkdir my-miniprogram
   cp -r dist/* my-miniprogram/
   cp -r cloudfunctions my-miniprogram/
   ```

4. **导入微信开发者工具**
   - 使用 `my-miniprogram` 目录作为项目目录导入

## 云函数配置

本项目包含以下云函数：

1. **users** - 用户管理（登录/注册、更新用户信息）
2. **rank** - 排行榜查询（获取前100名）
3. **updateUser** - 更新用户信息
4. **upload** - 头像上传

### 云函数部署步骤

1. 在微信开发者工具中，打开"云开发"控制台
2. 点击"云函数"
3. 右键点击每个云函数目录，选择"上传并部署：云端安装依赖"
4. 等待部署完成

## 环境配置

### 1. 配置云开发环境

在 `src/cloudbase/index.ts` 中修改：

```typescript
// 设置 USE_CLODBASE = true 启用云开发
export const USE_CLODBASE = true

// 配置云开发环境 ID
const cloudbase = cloudbaseSdk.init({
  env: 'your-env-id',  // 替换为你的云环境 ID
  // ...其他配置
})
```

### 2. 配置对象存储

确保已开通云开发对象存储服务，用于存储用户头像。

### 3. 配置 AppID

1. 在微信开发者工具中填写正确的 AppID
2. 在云开发控制台中开通云开发服务
3. 在 `project.config.json` 中更新 `appid` 字段

## 功能说明

### 主要功能

1. **开始游戏** - 从当前关卡开始，有倒计时限制
2. **自由模式** - 6种难度选择（3×3到8×8），无倒计时
3. **排行榜** - 显示前100名玩家
4. **用户设置** - 设置头像和昵称

### 游戏模式

- **正常模式**：有倒计时，记录游戏进度和积分
- **自由模式**：无倒计时，不记录进度

### 难度配置（自由模式）

1. 简单模式：3×3 网格
2. 入门模式：4×4 网格
3. 普通模式：5×5 网格
4. 困难模式：6×6 网格
5. 专家模式：7×7 网格
6. 大师模式：8×8 网格

## 注意事项

1. **首次导入后**，需要先配置云开发环境 ID
2. **云函数部署**：需要逐个部署云函数
3. **对象存储**：确保已开通对象存储服务
4. **图片源**：使用 Lorem Picsum 提供的随机图片
5. **缓存策略**：首次启动会预加载100张图片，约需10-20秒
