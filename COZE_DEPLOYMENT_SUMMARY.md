## 🎯 架构说明

### 扣子编程（开发环境）

- **作用**：前端代码开发、编译和构建
- **技术栈**：Taro + React + TypeScript + TailwindCSS
- **编译产物**：`dist/` 目录（小程序代码）

### 腾讯云开发（后端服务）

- **云函数**：6 个云函数处理业务逻辑
  - `login` - 用户登录
  - `updateUserInfo` - 更新用户信息
  - `updateHighestLevel` - 更新最高关卡
  - `getRankList` - 获取排行榜
  - `addPoints` - 添加积分
  - `consumePoints` - 消耗积分
- **云数据库**：存储用户数据（集合：`users`）
- **云存储**：存储用户头像（目录：`avatars`）

### 微信小程序（运行环境）

- **作用**：用户交互界面
- **开发工具**：微信开发者工具
- **发布平台**：微信小程序平台

## 🚀 快速部署（15 分钟）

### 步骤 1：安装依赖（3 分钟）

```bash
pnpm install
```

### 步骤 2：编译小程序（2 分钟）

```bash
pnpm build:weapp
```

### 步骤 3：下载编译产物（2 分钟）

下载 `dist` 和 `cloudfunctions` 目录到本地

### 步骤 4：用微信开发者工具打开（2 分钟）

导入项目，填写 AppID

### 步骤 5：配置腾讯云开发（3 分钟）

- 关联云开发环境：`cloudbase-8g1wqiy0823dea4a`
- 创建数据库集合 `users`

### 步骤 6：上传云函数（8 分钟）

上传 6 个云函数到腾讯云

### 步骤 7：测试（2 分钟）

测试登录和排行榜功能

## 🆘 常见问题

### 问题 1：编译失败

```bash
rm -rf node_modules pnpm-lock.yaml
pnpm install
pnpm build:weapp
```

### 问题 2：云函数调用失败

1. 检查云开发环境是否已关联
2. 检查云函数是否已上传
3. 检查环境 ID：`cloudbase-8g1wqiy0823dea4a`

### 问题 3：数据库操作失败

1. 检查数据库集合 `users` 是否已创建
2. 检查权限是否设置为"所有用户可读写"
