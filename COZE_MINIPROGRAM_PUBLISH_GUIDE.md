# 扣子编程环境小程序发布指南

## 🎯 发布流程概览

由于你使用的是扣子编程环境，代码部署在云端，发布微信小程序需要：

1. **从扣子编程导出代码**（或直接下载）
2. **在本地安装微信开发者工具**
3. **导入项目到微信开发者工具**
4. **按照标准流程发布**

---

## 📋 准备工作

### 1. 安装微信开发者工具

#### 下载地址
- [Windows 版](https://servicewechat.com/wxa-doc-acc/download/202010134663/96102/windows64/1.05.2106300/wechat_devtools_1.05.2106300_x64.exe)

2. 微信扫码登录
3. 进入「设置」→「项目设置」
4. 配置 AppID（如果没有，需要先注册）

## 🚀 发布方式

### 方式一：导出代码后本地发布（推荐）

#### Step 1: 从扣子编程导出代码
由于扣子编程环境代码在云端，你需要：

**选项 A：通过 Git 导出**
```bash
# 如果项目有 Git 仓库
git clone https://your-repo-url.git
cd project-directory
```

**选项 B：直接下载代码**
1. 在扣子编程环境中，将所有代码打包下载
2. 解压到本地目录

**选项 C：使用扣子提供的导出功能**（如果支持）
1. 在扣子编程界面找到「导出」按钮
2. 选择导出为微信小程序项目
3. 下载压缩包

#### Step 2: 安装依赖
```bash
# 进入项目目录
cd project-directory

# 安装依赖
npm install
# 或
pnpm install
```

#### Step 3: 编译小程序
```bash
# 编译微信小程序版本
npm run build:weapp
# 或
pnpm build:weapp

# 编译后的文件在 dist/ 目录
```

#### Step 4: 导入到微信开发者工具
1. 打开微信开发者工具
2. 点击「导入项目」
3. 选择项目目录：`dist/`
4. 填写 AppID
5. 点击「导入」

#### Step 5: 预览和测试
1. 在开发者工具中预览
2. 真机扫码测试
3. 确保所有功能正常

#### Step 6: 上传和发布
按照标准流程：
- 上传代码
- 提交审核
- 发布上线

---

### 方式二：在扣子环境中配置发布（如果支持）

#### 检查扣子是否支持自动发布
1. 登录扣子编程平台
2. 查看项目设置
3. 查找「发布」或「部署」选项

#### 如果支持自动发布
1. 配置微信小程序 AppID
2. 点击「发布到微信小程序」
3. 填写版本信息
4. 确认发布

## 🔧 具体操作步骤（推荐方式）

### Step 1: 获取代码

由于你已经在扣子编程中开发了代码，有几种方式获取：

#### 方法 1: 使用 Git（推荐）
如果你的项目连接了 Git 仓库：
```bash
# 克隆项目
git clone https://your-git-repo-url.git

# 进入项目目录
cd your-project-name

# 安装依赖
pnpm install

# 编译小程序
pnpm build:weapp
```

#### 方法 2: 手动下载
1. 在扣子编程环境中，复制所有代码
2. 在本地创建项目文件夹
3. 粘贴代码
4. 安装依赖和编译

---

### Step 2: 配置 AppID

在 `project.config.json` 中配置：
```json
{
  "appid": "你的微信小程序AppID",
  "projectname": "拼图小游戏",
  "compileType": "miniprogram"
}
```

**如果没有 AppID**：
1. 访问[微信公众平台](https://mp.weixin.qq.com)
2. 注册小程序账号
3. 获取 AppID
4. 在 `project.config.json` 中配置

---

### Step 3: 编译项目

```bash
# 编译微信小程序
pnpm build:weapp

# 查看编译结果
ls -la dist/
```

---

### Step 4: 导入微信开发者工具

1. 打开微信开发者工具
2. 点击「导入项目」
3. 选择目录：`dist/`（编译后的目录）
4. 填写项目信息：
   - 项目名称：拼图小游戏
   - AppID：你的 AppID
5. 点击「导入」

---

### Step 5: 预览和测试

#### 在开发者工具中测试
1. 点击「预览」按钮
2. 用微信扫码
3. 在真机上测试功能

#### 重点测试项
- [ ] 微信登录功能
- [ ] 游戏玩法（拖拽、交换）
- [ ] 排行榜功能
- [ ] 头像上传功能
- [ ] 各种机型适配

---

### Step 6: 上传代码

1. 点击右上角「上传」按钮
2. 填写版本号：`1.0.0`
3. 填写项目备注：
   ```
   拼图小游戏正式版
   - 支持微信登录
   - 100+ 关卡
   - 排行榜功能
   - 自定义头像
   ```
4. 点击「上传」

---

### Step 7: 发布体验版

1. 登录[微信公众平台](https://mp.weixin.qq.com)
2. 进入「开发」→「开发版本」
3. 找到刚上传的版本
4. 点击「选为体验版本」

#### 添加体验者
1. 进入「管理」→「成员管理」
2. 点击「项目成员/体验成员」
3. 搜索微信号
4. 添加为「体验成员」

---

### Step 8: 提交审核

1. 进入「管理」→「版本管理」→「开发版本」
2. 点击「提交审核」
3. 填写审核信息：
   ```
   功能介绍：
   这是一款经典的拼图小游戏，包含100+关卡，支持拖拽交换，
   包含排行榜和自定义头像功能，适合所有年龄段的玩家。

   服务类目：
   游戏 > 棋牌/益智

   页面截图：
   - 首页截图
   - 游戏界面截图
   - 排行榜截图
   ```

---

### Step 9: 等待审核并发布

1. 等待微信审核（1-7 个工作日）
2. 审核通过后，进入「审核版本」
3. 点击「发布」
4. 填写发布说明
5. 点击「确认发布」

## 📋 扣子编程环境特殊注意事项

### 1. 环境变量配置

由于你的代码部署在扣子环境，需要注意：

**开发环境**（扣子环境）：
```bash
PROJECT_DOMAIN=https://xxx.dev.coze.site
NODE_ENV=development
```

**生产环境**（发布到微信）：
```bash
PROJECT_DOMAIN=https://your-production-domain.com
NODE_ENV=production
```

**建议**：
- 发布前修改环境变量为生产环境
- 或者使用条件编译

### 2. API 域名配置

发布前需要在微信公众平台配置服务器域名：
1. 登录微信公众平台
2. 进入「开发」→「开发管理」→「开发设置」
3. 配置服务器域名：
   - request 合法域名：你的后端 API 域名
   - uploadFile 合法域名：你的后端 API 域名
   - downloadFile 合法域名：你的后端 API 域名

### 3. HTTPS 要求

微信小程序要求所有 API 必须使用 HTTPS：
- 开发环境可以使用 HTTP
- 生产环境必须使用 HTTPS

## 🔍 常见问题

### Q1: 扣子编程的代码如何导出？
**解决方法**：
- 如果有 Git 仓库，直接 clone
- 如果没有，手动复制代码到本地
- 联系扣子平台技术支持

### Q2: 编译后上传失败？
**可能原因**：
- AppID 未配置
- 项目目录选择错误
- 缺少必要文件

**解决方法**：
- 检查 `project.config.json`
- 选择 `dist/` 目录导入
- 确保所有必要文件都在 `dist/` 中

### Q3: 真机测试功能异常？
**可能原因**：
- API 域名未配置
- HTTPS 证书问题
- 权限未授权

**解决方法**：
- 在微信公众平台配置合法域名
- 确保使用 HTTPS
- 检查用户隐私授权

### Q4: 审核被拒怎么办？
**常见原因**：
- 隐私政策不完善
- 用户协议缺失
- 游戏内容违规

**解决方法**：
- 添加隐私政策和用户协议
- 检查游戏内容是否符合规范
- 根据审核意见修改后重新提交
