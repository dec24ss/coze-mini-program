# 版本更新日志

## v1.1.0 (2025-01-13)

### 新增功能
- ✨ 添加云开发支持（微信小程序云数据库、云函数）
- ✨ 添加排行榜功能（支持查看前100名玩家）
- ✨ 添加微信登录功能
- ✨ 添加用户信息设置（头像、昵称）
- ✨ 添加关卡解锁进度保存
- ✨ 添加游戏进度持久化保存
- ✨ 添加图片预加载功能
- ✨ 添加音效和震动反馈
- ✨ 添加设置界面（音效开关、震动开关）
- ✨ 添加自由模式（无倒计时，6种难度）

### 优化改进
- 🎨 统一使用蓝白色系配色方案
- 🎨 优化所有动画效果（过关弹窗、烟花、图块动画）
- 🎨 美化用户设置界面（头像呼吸光效、按钮光效）
- 🎨 美化难度选择页面（卡片式布局、渐变背景）
- ⚡ 优化图片加载性能（使用隐藏 Image 组件预缓存）
- ⚡ 优化排行榜渲染性能
- ⚡ 优化代码结构（提取公共逻辑、减少重复代码）

### 问题修复
- 🐛 修复首页白屏问题（图片预加载状态检查）
- 🐛 修复用户登录后头像显示问题
- 🐛 修复排行榜无法刷新问题（添加强制刷新和缓存降级）
- 🐛 修复关卡进度保存问题（使用本地存储）
- 🐛 修复云开发初始化错误（添加 USE_CLOUDBASE 开关）
- 🐛 修复图片显示问题（小程序端使用本地路径）

### 技术更新
- 🔧 添加 Zustand 5 状态管理
- 🔧 添加 @cloudbase/js-sdk 和 @cloudbase/node-sdk
- 🔧 添加 lucide-react-taro 图标库
- 🔧 添加网络请求封装（Network）
- 🔧 优化构建配置（添加云开发排除规则）

### 文档更新
- 📝 添加 README.md（项目说明、快速开始、开发规范）
- 📝 添加 EXPORT_GUIDE.md（导出部署指南）
- 📝 添加 IMPORT_GUIDE.md（导入部署指南）
- 📝 添加 WECHAT_RELEASE_GUIDE.md（微信小程序上传审核指南）
- 📝 添加 CLOUDBASE_MIGRATION.md（云开发数据迁移指南）
- 📝 添加 FIXES.md（问题修复日志）
- 📝 添加 CHANGELOG.md（版本更新日志）
- 📝 添加 export-for-preview.js（预览版导出脚本）

### 重要提示
- ⚠️ 在 Coze 环境中，云开发已禁用（USE_CLOUDBASE=false）
- ⚠️ 下载到本地后，需要修改 USE_CLOUDBASE=true 以启用云开发
- ⚠️ 使用测试号时，会出现云开发错误（可忽略）
- ⚠️ 需要正式 AppID 才能使用完整的云开发功能

## v1.0.0 (初始版本)

### 基础功能
- ✨ 拼图游戏核心功能（拖拽交换图块）
- ✨ 3×3 到 8×8 六种难度模式
- ✨ 加载页面（图片预加载）
- ✨ 首页（开始游戏、排行榜、设置）

- ✨ 难度选择页面（自由模式入口）

- Taro 4.1.9
- React 18
- TypeScript 5.4.5
- Tailwind CSS 4
- Zustand 5（状态管理）

## 版本说明

### 版本号规则
- 主版本号（Major）：重大架构变更或不兼容更新
- 次版本号（Minor）：新增功能或重要优化
- 修订号（Patch）：问题修复或小改进

### 更新频率
- 根据功能需求和问题修复情况不定期更新
- 重大版本更新会在 README.md 中提前说明
