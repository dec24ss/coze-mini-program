# 更新日志

所有重要的项目变更都将记录在此文件中。

## [v1.2.0] - 2026-03-02

### 🎉 主要更新
- 将数据库从本地 SQLite 迁移到 Supabase 云数据库
- 实现头像上传到对象存储功能
- 修复排行榜头像显示问题

### ✨ 新增功能
- **用户头像上传**：支持用户上传自定义头像，存储到对象存储
- **默认头像生成**：未设置头像的用户自动生成基于昵称的默认头像
- **排行榜优化**：显示前100名用户，按最高关卡和积分排序

### 🐛 修复问题
- 修复排行榜看不到其他用户头像的问题
- 修复本地头像路径无法跨设备访问的问题
- 修复头像URL有效性判断逻辑

### 🔧 技术改进
- 数据库从 SQLite 迁移到 Supabase PostgreSQL
- 使用 Supabase SDK 替代 Drizzle ORM
- 优化数据查询性能
- 完善错误处理和日志记录

### 📊 数据变更
- 排行榜已清空，准备接收新用户
- 数据库表结构优化（users 表）
- 新增 Row Level Security (RLS) 策略

### 🧪 测试
- 完成所有功能测试
- 数据库连接测试通过
- 用户CRUD操作测试通过
- 排行榜查询测试通过
- 头像上传测试通过
