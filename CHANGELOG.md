## v1.1.0 (2026-03-02)

### 🎉 新功能

### 🐛 Bug 修复
1. **部署环境图片服务返回 500 错误**
   - 修复了 veFaaS 部署环境中文件系统只读导致服务启动失败的问题
   - 添加 `useLocalImages` 标记，自动检测文件系统是否可写
   - 在文件系统只读时，自动使用 CDN URL 作为图片源

2. **用户上传的头像无法在排行榜显示**
   - 修复了头像上传后状态管理问题
   - 添加 `isAvatarUploaded` 标记，确保只有上传成功的头像才能保存
   - 优化 `updateUserInfo` 函数，使用后端返回的数据更新本地状态

3. **登录失败问题**
   - 修复了图片列表接口返回 500 错误导致的登录失败问题
   - 添加详细的调试日志，便于问题排查

## v1.0.0 (2026-02-XX)

### 🎉 初始版本
- 基于拼图游戏的小程序项目
- 包含加载、首页、游戏、关卡选择、排行榜五个界面
- 支持自由拖拽交换碎片位置
- 微信登录功能
- 排行榜和关卡解锁功能
- 基于 Supabase PostgreSQL 数据库
- 基于 NestJS 后端服务

### 核心功能
- 拼图游戏（3×3 到 7×7 难度）
- 关卡系统（100+ 关卡）
- 积分系统
- 用户排行榜
- 头像上传功能
- 音效和震动反馈
