## 📋 你需要完成的步骤

### 🔴 必须完成（关键步骤）

#### 1. 配置小程序 AppID
**文件**: `release-package/project.config.json`
```json
{
  "appid": "wx你的小程序AppID"
}
```

#### 2. 配置云开发环境
**文件**: `release-package/src/cloudbase/index.ts`
```typescript
const CLOUDBASE_ENV = '你的云开发环境ID'
```

#### 3. 运行一键部署脚本
**Mac/Linux**:
```bash
cd release-package
chmod +x deploy.sh
./deploy.sh
```

**Windows**:
```cmd
cd release-package
deploy.bat
```

#### 4. 开通云开发
1. 打开微信开发者工具
2. 导入 `release-package` 目录
3. 点击 "云开发" 按钮开通

#### 5. 部署云函数
1. 右键 `cloudfunctions/users` → "创建并部署"
2. 右键 `cloudfunctions/rank` → "创建并部署"
3. 右键 `cloudfunctions/updateUser` → "创建并部署"

#### 6. 创建数据库
1. 进入云开发控制台 → 数据库
2. 创建集合: `users` 和 `rankings`
3. 设置权限: "所有用户可读，仅创建者可读写"

#### 7. 上传代码
1. 点击 "上传" → 填写版本号 → 上传
2. 登录微信公众平台 → 版本管理 → 提交审核

## ⚠️ 重要提示

### BigInt 兼容性问题
- **问题**: 云开发 SDK 使用 BigInt，小程序 es2015 不支持
- **解决**: 压缩包中已处理，运行部署脚本时会自动启用云开发
- **注意**: 部署前必须运行脚本或手动启用云开发

### 需要你自己的
1. **小程序 AppID** - 在微信公众平台获取
2. **云开发环境 ID** - 开通云开发后获得
3. **微信开发者工具** - 下载安装

## 📞 常见问题速查
