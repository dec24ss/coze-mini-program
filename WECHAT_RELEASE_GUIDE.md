# 🚀 微信小程序上传审核完整指南

## 📋 前置检查清单

在开始上传审核之前，请确保以下事项都已完成：

- [ ] 小程序已在微信开发者工具中正常运行
- [ ] 云开发已开通并配置完成（如使用）
- [ ] 云函数已全部部署成功
- [ ] 数据库集合已创建
- [ ] 小程序 AppID 已配置（不是测试号）
- [ ] 已测试过所有核心功能
- [ ] 已准备好小程序图标和截图
- [ ] 已了解微信小程序审核规范

---

## 🎯 第一步：完善小程序信息

### 1.1 登录微信公众平台

1. 访问 [微信公众平台](https://mp.weixin.qq.com/)
2. 使用你的微信账号扫码登录
3. 选择你的小程序（如果有多个）

### 1.2 完善基本信息

在「设置」-> 「基本设置」中：

**小程序名称**
- 填写一个清晰的名称，如「趣味拼图」
- 名称不能与已有小程序重复
- 建议包含品牌或功能关键词

**小程序头像**
- 尺寸：144×144 像素
- 格式：PNG、JPG
- 大小：不超过 500KB
- 建议：简洁、清晰、易识别

**小程序介绍**
- 简要介绍小程序的功能
- 10-120 个字符
- 示例：「一款有趣的拼图小游戏，多种难度级别，等你来挑战！」

**服务类目**
- 选择合适的类目
- 拼图游戏建议选择：「工具 > 娱乐」或「游戏」
- 如果是游戏类目，需要游戏资质

---

## 🔧 第二步：配置服务器域名（如使用外部接口）

**注意**：如果你的小程序只使用云开发，不需要配置服务器域名！

在「开发」-> 「开发管理」-> 「开发设置」-> 「服务器域名」中：

如果使用了外部图片或接口，需要配置：

- **request 合法域名**：HTTPS 请求的域名
- **uploadFile 合法域名**：上传文件的域名
- **downloadFile 合法域名**：下载文件的域名
- **udp 合法域名**：UDP 通信的域名（一般不需要）

**拼图小程序主要需要配置**：
- `downloadFile 合法域名`: 添加 `picsum.photos`（图片源）
- `request 合法域名`: 如使用外部 API 也需要添加

---

## 📱 第三步：在开发者工具中上传代码

### 3.1 打开项目

1. 打开微信开发者工具
2. 确保打开的是你的拼图小程序项目
3. 确认 AppID 是正式的（不是测试号）

### 3.2 测试检查

1. 点击「编译」按钮
2. 在模拟器中测试所有功能：
   - [ ] 首页加载正常
   - [ ] 游戏功能正常
   - [ ] 关卡选择正常
   - [ ] 排行榜正常
   - [ ] 微信登录正常（如使用）
   - [ ] 音效播放正常

### 3.3 上传代码

1. 点击工具栏右上角的「上传」按钮
2. 填写版本号：如 `1.0.0`
3. 填写项目备注：如「首次提交审核」
4. 点击「上传」
5. 等待上传完成（通常几十秒）

---

## ✅ 第四步：在微信公众平台提交审核

### 4.1 进入版本管理

1. 登录 [微信公众平台](https://mp.weixin.qq.com/)
2. 进入「管理」-> 「版本管理」

### 4.2 提交审核

在「开发版本」中找到刚才上传的版本：

1. 点击「提交审核」按钮
2. 阅读并同意《审核须知》
3. 点击「下一步」

### 4.3 填写审核信息

**功能页面**（必须填写）：

选择你的小程序主要功能页面：
- 首页路径：`pages/index/index`
- 标题：「首页」

**补充材料和反馈**（可选但建议填写）：

- **功能描述**：简要描述小程序的功能
  示例：「这是一款拼图小游戏，用户可以选择不同难度级别进行游戏，完成关卡可以解锁更高难度，并在排行榜查看成绩。」

- **测试账号**：如果需要登录才能使用，提供测试账号
  （拼图游戏一般不需要）

- **备注**：其他需要说明的情况

### 4.4 确认提交

1. 检查所有信息填写正确
2. 点击「提交」
3. 确认提交成功

---

## ⏳ 第五步：等待审核

### 审核时间

- **首次审核**：通常 1-7 个工作日
- **更新版本**：通常 1-3 个工作日
- **加急审核**：有特殊情况可以申请，但条件严格

### 审核期间

- 不要修改代码或重新上传
- 保持云开发服务正常运行
- 可以准备下一版本的功能

### 审核结果通知

- 审核结果会通过「微信公众平台安全助手」公众号通知
- 也可以在微信公众平台查看审核状态

---

## 🎉 第六步：审核通过后发布

### 6.1 查看审核结果

1. 登录微信公众平台
2. 进入「管理」-> 「版本管理」
3. 查看审核结果

### 6.2 发布上线

如果审核通过：

1. 在「审核版本」中找到通过的版本
2. 点击「发布」按钮
3. 确认发布信息
4. 点击「确认发布」

### 6.3 发布完成

- 发布后几分钟内，用户就可以在微信中搜索到你的小程序
- 可以分享小程序给好友或群聊
- 可以生成小程序码

## ⚠️ 常见审核不通过原因及解决方案

### 1. 类目选择错误

**问题**：选择的类目与实际功能不符

**解决**：
- 拼图游戏建议选择「工具 > 娱乐」
- 如果使用「游戏」类目，需要《游戏版号》等资质
- 建议先用「工具」类目，审核通过后再考虑变更

### 2. 内容违规

**问题**：包含敏感内容、侵权内容等

**解决**：
- 确保图片来源合法（我们使用的 picsum.photos 是合法的）
- 不要包含政治、色情、暴力等内容
- 检查所有文本内容

### 3. 功能不完整

**问题**：功能无法正常使用、页面报错等

**解决**：
- 上传前充分测试所有功能
- 确保网络请求正常
- 检查云函数是否正常部署

### 4. 缺少必要资质

**问题**：某些类目需要特殊资质

**解决**：
- 选择不需要特殊资质的类目
- 如确实需要，按要求补充资质

### 5. 用户隐私问题

**问题**：收集用户信息但没有隐私协议

**解决**：
- 如果不收集用户信息，可以不用
- 如果收集，需要在「设置」-> 「基本设置」-> 「用户隐私保护指引」中配置

## 📸 审核需要的截图素材

建议提前准备以下素材（虽然审核时可能不需要，但发布后可以用于小程序介绍）：

### 小程序图标
- 尺寸：144×144 像素
- 格式：PNG
- 简洁、清晰、易识别

### 小程序截图（5张）
- 尺寸：iPhone 尺寸（1242×2208 像素）或 Android 尺寸
- 展示主要功能页面
- 截图要清晰、美观

### 建议的截图内容：
1. 首页（游戏选择界面）
2. 游戏进行中界面
3. 关卡选择界面
4. 排行榜界面
5. 游戏完成界面

## 💡 小贴士

1. **首次提交建议**：
   - 功能简单完整即可
   - 不要一次性上太复杂的功能
   - 先通过审核，再迭代更新

2. **审核期间**：
   - 不要修改代码
   - 保持服务正常运行
   - 可以准备下一版本

3. **审核不通过**：
   - 仔细阅读拒绝原因
   - 针对性修改
   - 修改后重新提交

4. **发布后**：
   - 监控用户反馈
   - 及时修复 bug
   - 定期更新功能
