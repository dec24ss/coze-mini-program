# 📱 微信真机预览指南

## 🎯 关于 Coze 环境中的微信扫码预览

**重要提示**：在 Coze 编程环境中，**无法直接**通过微信扫码预览小程序真机效果。

---

## 💡 推荐方案

### 方案一：下载项目到本地预览（推荐）⭐

这是最简单、最可靠的方式：

#### 步骤 1：在 Coze 中完成代码开发
- ✅ 在 Coze 中编写和修改代码
- ✅ 确保 Web 版本运行正常

#### 步骤 2：下载项目到本地
1. 在 Coze 界面找到"下载"或"导出"功能
2. 下载整个项目到你的电脑

#### 步骤 3：本地预览
1. 解压下载的项目
2. 打开终端，进入项目目录
3. 安装依赖：
   ```bash
   pnpm install
   ```
4. 启动小程序开发模式：
   ```bash
   pnpm dev:weapp
   ```
5. 打开微信开发者工具
6. 导入项目，选择 `dist` 目录
7. 点击"预览"按钮，扫码预览

---

### 方案二：使用 Web 版本测试（快速）

在 Coze 环境中，你可以：

1. **直接预览 Web 版本**
   - Coze 会自动启动 Web 服务器
   - 在浏览器中测试功能
   - 虽然不是真机，但可以测试大部分逻辑

2. **确认功能正常后，再下载到本地**
   - Web 版本测试通过
   - 下载到本地做真机测试

---

## 🔧 尝试小程序构建（可选）

如果你想尝试在 Coze 中构建小程序（可能会有警告）：

### 步骤 1：尝试构建
```bash
pnpm build:weapp
```

**注意**：可能会遇到 BigInt 兼容性警告，但可能仍然可以生成可用的 `dist` 目录。

### 步骤 2：如果构建成功
- 检查 `dist` 目录是否生成
- 如果生成了，可以下载 `dist` 目录
- 在微信开发者工具中打开 `dist` 目录

---

## 📋 完整开发流程建议

### 阶段 1：Coze 环境开发
1. ✅ 在 Coze 中编写代码
2. ✅ 使用 Web 版本测试功能
3. ✅ 确认代码逻辑正确

## 🆘 常见问题

**Q: Coze 中能直接微信扫码预览吗？**
A: 不能，需要下载到本地，使用微信开发者工具。

**Q: 小程序构建失败怎么办？**
A: 这是正常的，建议下载到本地构建，本地环境兼容性更好。

**Q: Web 版本和小程序版本有区别吗？**
A: 大部分功能是一样的，但某些原生功能（如微信登录、支付等）只能在小程序中测试。

**Q: 如何快速测试？**
A: 先用 Web 版本测试主要逻辑，确认没问题后再下载到本地做真机测试。
