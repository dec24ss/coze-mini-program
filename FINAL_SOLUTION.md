## 💻 下载到本地后的使用

### 步骤 1：下载项目
在 Coze 界面找到"下载"或"导出"功能，下载整个项目

### 步骤 2：本地启用云开发
下载到本地后，做以下修改：

#### 修改 1：恢复云开发 Store
```bash
# 在本地项目中
cp src/stores/userStore-cloudbase.ts.backup src/stores/userStore-cloudbase.ts
```

然后修改所有页面，将 `useUserStore` 改回 `useUserStoreCloudbase`

#### 修改 2：恢复云开发初始化
修改 `src/pages/loading/index.tsx`，恢复云开发初始化代码

#### 修改 3：启用云开发
修改 `src/cloudbase/index.ts`，将 `USE_CLOUDBASE` 改为 `true`

### 步骤 3：本地真机预览
```bash
# 安装依赖
pnpm install

# 启动小程序开发模式
pnpm dev:weapp

# 用微信开发者工具打开 dist 目录
# 点击"预览"，扫码真机测试
```

### 步骤 4：部署云函数
参考 `QUICK_DEPLOY.md`，使用云开发控制台部署云函数
