# 云函数快速部署指南

## 🚀 方法一：微信开发者工具部署（推荐）

### 步骤 1：准备工作

确保你已经：
- ✅ 打开微信开发者工具
- ✅ 打开本项目
- ✅ 看到左侧文件树中有 `cloudfunctions` 文件夹（带云朵图标☁️）

### 步骤 2：开通云开发

1. 点击微信开发者工具顶部的 **"云开发"** 按钮
2. 如果是第一次，点击 **"开通"**
3. 选择环境：`cloudbase-8g1wqiy0823dea4a`
4. 等待开通完成（可能需要1-2分钟）

### 步骤 3：部署 users 云函数

1. 在左侧文件树中，展开 `cloudfunctions` 文件夹
2. 右键点击 `users` 文件夹
3. 选择 **"上传并部署：云端安装依赖"**
   - 如果没有这个选项，选择 **"上传并部署"**
   - 然后在弹出框中选择 **"云端安装依赖"**
4. 等待上传完成，看到"上传成功"提示

### 步骤 4：部署 rank 云函数

1. 右键点击 `rank` 文件夹
2. 选择 **"上传并部署：云端安装依赖"**
3. 等待上传完成

### 步骤 5：部署 updateUser 云函数

1. 右键点击 `updateUser` 文件夹
2. 选择 **"上传并部署：云端安装依赖"**
3. 等待上传完成

### 步骤 6：验证部署

1. 点击顶部 **"云开发"** 按钮
2. 点击 **"云函数"** 标签
3. 你应该能看到三个云函数：`users`、`rank`、`updateUser`

## 🌐 方法二：云开发控制台部署（最可靠）

### 步骤 1：访问控制台

1. 打开浏览器，访问：https://console.cloud.tencent.com/tcb
2. 登录你的腾讯云账号
3. 选择环境：`cloudbase-8g1wqiy0823dea4a`

### 步骤 2：创建 users 云函数

1. 点击左侧菜单 **"云函数"**
2. 点击 **"新建"** 按钮
3. 填写信息：
   - 函数名称：`users`
   - 运行环境：Node.js 16（或最新版本）
   - 创建方式：**空白函数**
4. 点击 **"确定"**

### 步骤 3：复制 users 函数代码

1. 在云函数列表中，点击刚创建的 `users`
2. 点击 **"函数代码"** 标签
3. 删除编辑器中的所有默认代码
4. 在你的项目中打开文件：`cloudfunctions/users/index.js`
5. 全选复制（Ctrl+A 然后 Ctrl+C）
6. 粘贴到云开发控制台的编辑器中（Ctrl+V）
7. 点击 **"保存"** 按钮
8. 点击 **"保存并安装依赖"**（如果有这个按钮）

### 步骤 4：测试 users 云函数

1. 点击 **"测试"** 标签
2. 在测试参数中输入：
```json
{
  "openid": "test_001",
  "nickname": "测试用户",
  "avatar_url": ""
}
```
3. 点击 **"测试运行"**
4. 应该能看到返回结果：
```json
{
  "code": 200,
  "msg": "success",
  "data": { ... }
}
```

### 步骤 5：创建 rank 云函数

重复步骤 2-4：
- 函数名称：`rank`
- 代码文件：`cloudfunctions/rank/index.js`
- 测试参数：
```json
{
  "limit": 100
}
```

### 步骤 6：创建 updateUser 云函数

重复步骤 2-4：
- 函数名称：`updateUser`
- 代码文件：`cloudfunctions/updateUser/index.js`
- 测试参数：
```json
{
  "openid": "test_001",
  "nickname": "新昵称",
  "highest_level": 5,
  "points": 100
}
```

## 📊 创建数据库集合

### 步骤 1：创建集合

1. 在云开发控制台，点击左侧 **"数据库"**
2. 点击 **"新建集合"**
3. 集合名称：`users`
4. 点击 **"确定"**

### 步骤 2：配置权限

1. 点击刚创建的 `users` 集合
2. 点击 **"权限设置"** 标签
3. 选择 **"所有用户可读，仅创建者可写"**
4. 点击 **"确定"**

## 🆘 常见问题

**Q: 微信开发者工具看不到云函数文件夹？**
A: 确保 project.config.json 中有 "cloudfunctionRoot": "./cloudfunctions/"，然后重启微信开发者工具

**Q: 云函数上传失败？**
A: 检查网络连接，确保云开发环境已开通，尝试使用"云端安装依赖"模式

**Q: 云函数测试失败？**
A: 查看云函数日志，确认代码已正确复制，检查数据库权限配置

**Q: 数据库访问被拒绝？**
A: 确认数据库权限设置为"所有用户可读，仅创建者可写"
