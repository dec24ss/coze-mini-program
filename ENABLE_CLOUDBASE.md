# 🚀 云开发启用完整指南

## 📋 前提条件

在开始之前，请确保：
- ✅ 已下载 `puzzle-miniprogram-final.tar.gz` 到本地
- ✅ 已解压项目文件
- ✅ 已安装微信开发者工具
- ✅ 已有微信小程序账号（或使用测试号）

---

## 第一步：开通云开发

### 1.1 导入项目
1. 打开微信开发者工具
2. 点击「+」号或「导入项目」
3. 选择解压后的项目文件夹
4. AppID 选择你的小程序 AppID（或选择测试号）
5. 点击「导入」

### 1.2 开通云开发
1. 在微信开发者工具顶部工具栏，点击「云开发」按钮
2. 如果是首次使用，会弹出开通提示
3. 点击「开通」
4. 选择「按量付费」（有免费额度）
5. 填写环境名称，例如：`puzzle-game`
6. 点击「确定」
7. 等待环境创建完成（约 1-2 分钟）

### 1.3 获取环境 ID
1. 云开发开通后，会自动打开云开发控制台
2. 在控制台顶部可以看到环境名称和环境 ID
3. **环境 ID** 格式类似：`cloudbase-xxxxx`
4. **复制这个环境 ID**，后面会用到

---

## 第二步：创建数据库集合

### 2.1 打开数据库
1. 在云开发控制台左侧菜单，点击「数据库」
2. 点击「+」号创建集合

### 2.2 创建 users 集合
1. 集合名称：`users`
2. 点击「确定」
3. 点击刚创建的 `users` 集合
4. 点击「权限设置」
5. 选择「所有用户可读，仅创建者可写」
6. 点击「保存」

### 2.3 创建 rank 集合
1. 再次点击「+」号创建集合
2. 集合名称：`rank`
3. 点击「确定」
4. 点击刚创建的 `rank` 集合
5. 点击「权限设置」
6. 选择「所有用户可读，仅创建者可写」
7. 点击「保存」

---

## 第三步：修改配置文件

### 3.1 找到配置文件
在项目文件夹中找到：`src/cloudbase/index.ts`

### 3.2 修改云开发开关
打开文件，找到第 5 行：
```typescript
export const USE_CLOUDBASE = false
```
改为：
```typescript
export const USE_CLOUDBASE = true
```

### 3.3 填入环境 ID
找到第 2 行：
```typescript
const CLOUDBASE_ENV = 'cloudbase-8g1wqiy0823dea4a'
```
改为你的环境 ID：
```typescript
const CLOUDBASE_ENV = '你的云开发环境ID'  // 例如：cloudbase-abc123
```

### 3.4 保存文件
保存修改后的文件

---

## 第四步：部署云函数

### 4.1 找到云函数目录
在微信开发者工具左侧文件树中：
1. 找到 `cloudfunctions` 文件夹
2. 展开可以看到三个云函数：
   - `users`
   - `rank`
   - `updateUser`

### 4.2 部署 users 云函数
1. 右键点击 `users` 文件夹
2. 选择「上传并部署：云端安装依赖」
3. 等待上传和部署完成（约 1-2 分钟）
4. 看到「上传成功」提示

### 4.3 部署 rank 云函数
1. 右键点击 `rank` 文件夹
2. 选择「上传并部署：云端安装依赖」
3. 等待部署完成

### 4.4 部署 updateUser 云函数
1. 右键点击 `updateUser` 文件夹
2. 选择「上传并部署：云端安装依赖」
3. 等待部署完成

---

## 第五步：重新构建项目

### 5.1 安装依赖（如果还没有）
在项目根目录打开终端：
```bash
pnpm install
```

### 5.2 构建小程序版本
```bash
pnpm build:weapp
```

### 5.3 等待构建完成
看到构建成功的提示后，继续下一步

---

## 第六步：测试验证

### 6.1 刷新项目
在微信开发者工具中：
1. 点击「编译」按钮
2. 等待项目重新加载

### 6.2 测试微信登录
1. 在首页点击「开始游戏」
2. 应该会弹出微信登录授权
3. 同意授权后，应该显示用户信息

### 6.3 测试排行榜
1. 进入游戏并完成一关
2. 点击「排行榜」按钮
3. 应该能看到你的成绩

### 6.4 检查云开发数据
1. 打开云开发控制台
2. 点击「数据库」
3. 查看 `users` 集合，应该有用户数据
4. 查看 `rank` 集合，应该有排行榜数据

---

## ✅ 验证成功标志

- [ ] 微信登录正常，显示用户信息
- [ ] 游戏进度能保存（关闭小程序后再打开，进度还在）

## 🆘 常见问题

### Q1: 点击「云开发」按钮没反应？
**A**: 确保你已经登录微信开发者工具，并且选择了正确的 AppID。

### Q2: 云函数部署失败？
**A**: 
1. 检查网络连接
2. 确保云开发已开通
3. 稍等几分钟后重试

### Q3: 微信登录失败？
**A**: 
1. 检查 `USE_CLOUDBASE` 是否改为 `true`
2. 检查环境 ID 是否正确
3. 查看控制台错误日志

### Q4: 排行榜不显示？
**A**: 
1. 确认 `rank` 云函数已部署
2. 先玩一局游戏提交成绩
3. 查看云开发数据库是否有数据

### Q5: 数据没有保存？
**A**: 
1. 检查数据库权限设置
2. 查看云函数日志
3. 确认云函数部署成功
