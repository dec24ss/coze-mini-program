## 配置步骤

### 通过 Vercel 网页控制台配置

1. 访问 [Vercel](https://vercel.com) 并登录
2. 进入项目设置页面
3. 点击 **Settings** → **Environment Variables**
4. 点击 **Add New** 添加环境变量
5. 按照上述清单逐个添加

### 快速复制命令（CLI 方式）

```bash
# 登录 Vercel
vercel login

# 进入项目目录
cd /workspace/projects

# 链接项目
vercel link

# 添加环境变量
vercel env add COZE_SUPABASE_URL production
# 输入：https://egutrdawrbziyklwnuov.supabase.co

vercel env add COZE_SUPABASE_ANON_KEY production
# 输入：eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJpc3MiOiJzdXBhYmFzZSIsInJlZiI6ImVndXRyZGF3cmJ6aXlrbHdudW92Iiwicm9sZSI6ImFub24iLCJpYXQiOjE3NzI0MjUwMTEsImV4cCI6MjA4ODAwMTAxMX0.PKGOVO2ItYvesw714VRdGxuTjGgNw5WU2LHTDLObQfs

vercel env add PROJECT_DOMAIN production
# 输入：https://dec24ss-coze-mini-program.vercel.app

# 部署
vercel --prod
```

## 常见问题

### Q1: 部署失败，提示环境变量未设置

**A**: 检查是否在所有环境（Production, Preview, Development）都配置了环境变量。

### Q2: Supabase 连接失败

**A**: 检查 `COZE_SUPABASE_URL` 和 `COZE_SUPABASE_ANON_KEY` 是否正确。

### Q3: 如何更新环境变量？

**A**: 在 Vercel 项目设置页面修改，然后重新部署。
